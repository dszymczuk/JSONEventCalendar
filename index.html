JSONEventCalendar

Description

JSONEventCalendar is jQuery plugin to create calendar which shows events in JSON format. Each event has element like: title, date and time, type, link and more.

You can choose how calendar works - show only calendar, list of events of both. You can set own date format to show on list of events, choose colors of evetns and more - check feature list.

  • Show calendar, list of events or both
